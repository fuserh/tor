PrivChat

Tor Project がお届けする、テクノロジー、人権、
、そしてインターネットの自由についての会話

PrivChat は、Tor Project への寄付を募るために開催される募金イベント シリーズです。 PrivChat を通じて、コミュニティとのチャットのために専門家を招集することにより、テクノロジー、人権、およびインターネットの自由で起こっていることに関連する重要な情報をお届けします。

PrivChat への参加は無料です。これらのイベントに価値を得て、Tor を気に入っていただけた場合は、毎月の寄付をご検討ください。確実で予測可能なサポートは、Tor が強力で安定した状態を維持するための最善の方法です。

PrivChat の目標は、双方向のサポートシステムを構築することです。プライバシー、テクノロジー、人権について考え、活動している最前線の人たちの情報にアクセスできます。また、Tor Project は皆様のご支援の結果、開発においてより機敏になり、増加する監視と検閲の脅威に迅速に対応できるようになります (そして、より多くの PrivChat をホストできるようになります) ！

第6章 - プライバシーは人権です

12/15 ∙ 19:00 UTC ∙ 14:00 Eastern ∙ 11:00 Pacific ∙ @torproject YouTube channel

見る

プライバシーとは、私たちを人間たらしめているもの、つまり日々の行動、性格、恐怖、人間関係、そして弱さなどを守ることです。誰もがプライバシーを守る権利があります。国連は1948年の世界人権宣言でプライバシーを人権として成文化しました。しかし、政府や企業、その他の強力な組織は、監視、追跡を通じてプライバシー権の行使を日常的に妨害しています。

活動家、人権擁護活動家、マイノリティ、変革のために抗議する人々は、こうした監視の対象となることが多いため、オンラインにおけるプライバシーと匿名性の重要性について独自の視点を持っています。

今回の PrivChat では、活動家として、また活動家グループと一緒に活動した経験を持つパネリストの方々を集め、監視とプライバシーに関する経験についてお話していただきます。

スウェーデン国際開発協力庁シニアプログラムスペシャリストの Ali Gharavi、アーティスト、活動家、Pussy Riot の創設メンバーの Nadya Tolokonnika、Calyx Institute 創設者兼事務局長の Nicholas Merrill と、監視の経験や、なぜプライバシーのために闘うことがすべての人の人権を保障する上で重要なのかについて議論しましょう。

EFF の事務局長である Cindy Cohn が司会兼モデレーターとして参加します。

Cindy Cohn

電子フロンティア財団 事務局長

Cindy Cohn は、電子フロンティア財団の事務局長です。2000年から2015年まで、EFF の法務部長および法務顧問を務めました。1993年には、暗号技術に関する米国の輸出規制に対する合衆国憲法修正第一条の異議申し立てを成功させた Bernstein 対司法省訴訟の主任弁護士を務めました。特に、Cohn は TheNonProfitTimes の2020年の Power&Influence TOP 50リストに選出され、2018年には Forbes 誌のアメリカのテクノロジーにおける女性トップ50にも選出されました。2013年、The National Law Journal は Cohn をアメリカで最も影響力のある100人の弁護士の一人に挙げ、こう評しました。「ビッグブラザーが見ているなら、Cindy Cohn に気をつけた方がいい」

第5章 Pegasus から身を守る

Likhita、Etienne Maynier、John Scott-Railton と共に。司会は Roger Dingledine です。

毎年、政府、法執行機関、軍隊、および企業は、悪意のあるスパイウェア (ユーザーのデバイスに密かに侵入し、攻撃者が検出されずに内部のコンテンツを閲覧できるように設計されたソフトウェア) の構築と購入に数十億ドルを投資しています。 今年、Pegasus Project は、NSOグループが構築した「Pegasus」と呼ばれるこの種のスパイウェアを用いた攻撃者が、企業幹部、政治家、ジャーナリスト、人権活動家など、50カ国以上の数千人が所有する携帯電話を標的にしていたことを明らかにしました。 今回の PrivChat では、Amnesty International の Likhita と Etienne Maynier、Citizen Lab の John Scott-Railton と一緒に、人々が自分自身を守る方法、このような悪用を止めるために私たちが支援できる組織、そして私たちが信頼できるより安全でよりプライベートなソフトウェアに取り組んでいるのは誰かについて議論します。

第4章 - オニオンルーティングの25周年記念

Roger Dingledine、Nick Mathewson、Paul Syverson と共に。司会は Gabriella Coleman です。

Tor でオニオンルーティングの 25 周年を祝いましょう！ 2021年5月31日は、英国のケンブリッジで開催されたアイザック・ニュートン研究所の第1回情報秘匿ワークショップで、オニオンルーティングが初めて公開されてから25年目にあたります。

この特別な瞬間を祝うために、オニオンルーティングの始まりから、このアイデアがどのように Tor になり、Tor Project が最終的にどのようになったかについて、私たちと一緒に話しませんか？最初のオニオンルーティング論文の著者の一人である Paul Syverson と、Tor Project の共同創設者 Roger Dingledine、Nick Mathewson が参加する予定です。

米国海軍研究所 (NRL) のオニオンルーティングネットワークの最初の頃を振り返ってみます。（当時、オニオンルーター接続は、Tor の現在の3ノードデザインではなく、5ノードを経由していました！）オニオンルーティングのコンセプトがNRLで生まれたことは周知の事実ですが（詳しくは私たちの歴史ページより）、Tor の始まりからの25年の歩みについてお知らせしたいことはまだまだたくさんあります。

第3章 - Tor の人権促進

Alison Macrina、Berhan Taye、Ramy Raoof と共に。司会は Edward Snowden です。

Tor Browser の主な使命は、フリー・オープンソースの匿名性・プライバシーに関する技術を創造、開発することで、人権と自由を発展させることにあります。私たちのテクノロジー、すなわち Tor ネットワークと Tor Browser は、さまざまな用途で利用されています。Tor は、不正に関する情報、社会にとって不可欠な情報を、身元を明かすことなく安全に明るみに出す手段を必要とする内部告発者によって利用されています。Tor は、権威主義的な政府に抵抗し、人権を守るために世界中の活動家によって利用されています。彼らの安全と匿名性のためだけでなく、インターネット検閲を回避し、彼らの声を届けるためでもあります。Tor は、何百万人もの人々が、特権の有無にかかわらず、オンラインで自分自身を守ることを可能にします。第3回の PrivChat では、実際の Tor ユーザーをお招きし、Tor が彼らにとってどのように重要であったか、また世界中の人権と自由を守るためにどのような活動をしているかについてお話ししていただきます。

第2章 - 検閲回避の良い面、悪い面

Felicia Anthonio、Vrinda Bhandari、Cecylia Bocovich、Arturo Filastò と共に。司会は Cory Doctorow です。

年々、インターネット検閲は世界的に増加しています。政府や民間企業は、ネットワークレベルのブロックから国全体ののインターネット遮断に至るまで、情報を制限し、人々の間の接続を防ぐための強力なツールを持っています。多くの人々、グループ、組織が、インターネット検閲を調査、測定し、これに対抗するために革新的な取り組みを行っており、何百万人もの人々がより定期的かつ安全にインターネットに接続できるよう支援しています。これらの成功にもかかわらず、私たちは検閲メカニズムに数十億ドルを費やす資金力のある相手に直面しており、軍拡競争は続いています。PrivChat with Tor の第2回では、検閲回避の最前線で起こっている良いこと、悪いことについてお話します。検閲技術がますます洗練され、国家間で売買されるようになった世界では、それを測定し、それを回避するツールを構築するための創造性も、人々が反撃する意欲も高まっています。しかし、それで十分なのでしょうか？インターネットの自由のために闘う人々や組織が直面している障壁は何でしょうか？

第1章 - 2020年のオンラインプライバシー: アクティビズムと COVID-19

Carmela Troncoso、Daniel Kahn Gillmor、Matt Mitchell と共に。司会は Roger Dingledine です。

COVID-19 のパンデミックが世界のほとんどの国を襲ったとき、多くの政府はパンデミックと闘うためにウイルスの拡散を追跡する技術を探しました。接触者追跡の慣行と技術は、プライバシーに関する多くの疑問、特に人々のプライバシーを尊重しながらウイルスを追跡することは可能かという疑問を提起しました。 現在、米国での組織的な人種差別に対する反発、そしてそれに続く世界各地での抗議運動の中で、接触者の追跡、プライバシー、監視に関する中心的な疑問が重大なものとなっています。ウイルスの追跡に使われている技術は、抗議者の追跡にも使えるのでしょうか？そうなるのでしょうか？ 初めての PrivChat では、Tor Project が3人の素晴らしいゲストをお招きし、この文脈におけるプライバシーについてお話ししていただきます。
