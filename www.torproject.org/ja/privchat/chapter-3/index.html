PrivChat は、Tor Project への寄付を募るために開催される募金イベント シリーズです。 PrivChat を通じて、コミュニティとのチャットのために専門家を招集することにより、テクノロジー、人権、およびインターネットの自由で起こっていることに関連する重要な情報をお届けします。

第3章 - Tor の人権促進

見る

Tor Browser の主な使命は、フリー・オープンソースの匿名性・プライバシーに関する技術を創造、開発することで、人権と自由を発展させることにあります。私たちのテクノロジー、すなわち Tor ネットワークと Tor Browser は、さまざまな用途で利用されています。Tor は、不正に関する情報、社会にとって不可欠な情報を、身元を明かすことなく安全に明るみに出す手段を必要とする内部告発者によって利用されています。Tor は、権威主義的な政府に抵抗し、人権を守るために世界中の活動家によって利用されています。彼らの安全と匿名性のためだけでなく、インターネット検閲を回避し、彼らの声を届けるためでもあります。Tor は、何百万人もの人々が、特権の有無にかかわらず、オンラインで自分自身を守ることを可能にします。第3回の PrivChat では、実際の Tor ユーザーをお招きし、Tor が彼らにとってどのように重要であったか、また世界中の人権と自由を守るためにどのような活動をしているかについてお話ししていただきます。

ホスト

Edward Snowden

報道の自由財団会長

Edward Snowden は米国市民であり、元諜報機関の職員であり、内部告発者です。彼が開示した文書は、NSA とその国際的な諜報機関のパートナーによる秘密の大量監視プログラムと能力についての、重要な情報を社会に提供しました。これらの事実が明らかになったことで、世界中でプライバシー侵害とデジタルセキュリティに関する前例のない注目が集まり、この問題に関する世界的な議論につながりました。

参加者

Alison Macrina

Library Freedom Project 創設者

Alison Macrina は図書館員であり、現実世界で知的自由とプライバシーの価値観を生きる図書館職員のための実践コミュニティである Library Freedom Project の創設者です。彼女の研究は、監視が図書館コミュニティや図書館員の仕事にどのように影響するか、監視が他の社会的正義の問題や知的自由とどのように関係するか、そして私たちの技術の内部の力のダイナミクスに焦点を当てています。Alison と Library Freedom Project は、Free Software Foundation の社会的利益プロジェクト賞、Library Journal の Movers and Shakers 賞、New York Library Association の知的自由賞、LITA/Library Hi Tech 賞を受賞しています。

Berhan Taye

アフリカポリシーマネージャー兼グローバルインターネットシャットダウン対抗担当、Access Now

Berhan Taye は、テクノロジーと社会と社会的正義の関係を研究している研究者です。彼女は、Access Now のアフリカポリシーマネージャー兼グローバルインターネットシャットダウン対抗担当です。彼女は、世界中の220以上のメンバー組織の協会とともに、インターネットの遮断を阻止する世界的なキャンペーンである #KeepItOn キャンペーンを率いてきました。Access Now に参加する前は、MoreThanCode.cc レポートを作成した Technology for Social Justice Field Scan プロジェクトの研究者でした。

Ramy Raoof

セキュリティラボ技術者、Amnesty International

Ramy Raoof は、技術者およびプライバシー研究者で、最近の研究では、人権擁護活動家や NGO に対する標的型デジタル攻撃の研究、中東や中央アメリカの活動家とのデジタルセキュリティプロトコルの開発、監視と検閲に対抗する能力開発に焦点を当てています。Ramy は、Amnesty International のセキュリティラボの戦術技術者であり、Tor Project の理事でもあります。Amnesty に入る前は、Egyptian Initiative for Personal Rights (EIPR) の上級技術者を務めていました。Ramy は、2017年に Access Now から「Heroes of Human Rights and Communications Surveillance」という国際賞を受賞し、2016年5月には、デジタルセキュリティとプライバシーの分野での活動が認められ、Deutsche Welle から「Bobs 賞 - Best of Online Activism 」を受賞しました。Twitter: https://twitter.com/RamyRaoof
