PrivChat

Tor Project がお届けする、テクノロジー、人権、
、そしてインターネットの自由についての会話

PrivChat は、Tor Project への寄付を募るために開催される募金イベント シリーズです。 PrivChat を通じて、コミュニティとのチャットのために専門家を招集することにより、テクノロジー、人権、およびインターネットの自由で起こっていることに関連する重要な情報をお届けします。

第1章 - 2020年のオンラインプライバシー: アクティビズムと COVID-19

見る

COVID-19 のパンデミックが世界のほとんどの国を襲ったとき、多くの政府はパンデミックと闘うためにウイルスの拡散を追跡する技術を探しました。接触者追跡の慣行と技術は、プライバシーに関する多くの疑問、特に人々のプライバシーを尊重しながらウイルスを追跡することは可能かという疑問を提起しました。 現在、米国での組織的な人種差別に対する反発、そしてそれに続く世界各地での抗議運動の中で、接触者の追跡、プライバシー、監視に関する中心的な疑問が重大なものとなっています。ウイルスの追跡に使われている技術は、抗議者の追跡にも使えるのでしょうか？そうなるのでしょうか？ 初めての PrivChat では、Tor Project が3人の素晴らしいゲストをお招きし、この文脈におけるプライバシーについてお話ししていただきます。

ホスト

Roger Dingledine

Roger Dingledine は、オンライン上の追跡、検閲、監視から人々を守るために、フリーでオープンソースのソフトウェアを開発する非営利団体 Tor Project の代表＆共同設立者です。
一つの帽子をかぶって、Roger は多くの大陸のジャーナリストや活動家と協力し、彼らが直面している脅威を理解し、対抗する手助けをしています。同氏はオンライン匿名性分野の主任研究者であり、Tor 関連のトピックに取り組む学術研究者の調整や指導を行っています。2002年以来、毎年国際的なプライバシー強化技術シンポジウム (PETS) の開催を支援しています。
彼の功績として、MITテクノロジーレビュー誌の35歳以下の革新者トップ35に選ばれ、Usenix Security の「Test of Time」賞を受賞した Tor のデザインペーパーを共同執筆し、Foreign Policy 誌のグローバル思想家トップ100に選出されています。

参加者

Carmela Troncoso

Carmela Troncoso は EPFL (スイス) の助教授で、SPRING Lab を率いています。ビーゴ大学で電気通信工学の修士号 (2006年) を、KU Leuven で工学の博士号 (2011年) を取得しています。EPFL に着任する前は、IMDEA Software Institute (スペイン) で2年間教員を務めていました。セキュリティとプライバシーに関する Gradient のテクニカルリードは、業界と緊密に連携して、安全でプライバシーに配慮したソリューションを市場に4年間提供しています。COSIC グループのポスドク研究員でもあります。
Carmela の研究は、セキュリティとプライバシーに重点を置いています。彼女の論文「Design and Analysis methods for Privacy Technologies」は欧州情報処理数学研究コンソーシアム最優秀博士論文賞を受賞。プライバシーエンジニアリングに関する研究は、CNIL-INRIA プライバシー保護賞2017を受賞しました。彼女はセキュリティ(ACM Conference on Computer Security や USENIX Security Symposium など)とプライバシー (プライバシー強化技術) の最も権威ある場所で定期的に発表しています。

Daniel Kahn Gillmor

Daniel Kahn Gillmor は ACLU の Speech, Privacy, and Technology Project のシニアスタッフテクノロジストであり、技術インフラが社会を形成し、市民の自由に影響を与える仕組みに焦点を当てています。
フリーソフトウェア開発者であり、Debian Project のメンバーとして、情報が豊富な環境の可能性を形作る基本的なツールに貢献しています。
IETF の参加者として、プライバシーとセキュリティのために設計および最適化された新世代のネットワークおよび暗号プロトコルの開発を促進しています。プライバシー、正義、言論の自由、データ主権のための反監視の提唱者でもあります。ブラウン大学コンピューターサイエンスプログラム卒業。

Matt Mitchell

Matt Mitchell はハッカーであり、The Ford Foundation の技術研究員です。Matt は、Ford Foundation の BUILD および Technology and Society チームと協力して、デジタルセキュリティ戦略、技術支援サービス、および Ford Foundation の助成パートナー向けの安全とセキュリティ対策を開発しています。
ハッカー、開発者、オペレーショナルセキュリティトレーナー、セキュリティ研究者、データジャーナリストとしてのデジタルスキルを善のために使うことにコミットしている Matt は、テクノロジーと公共の福祉の交差点でさまざまな立場で働いてきました。以前は、Tactical Tech (Tactical Technology Collective としても知られています) の Digital Safety & Privacy のディレクターでした。Matt は、プライバシーに関する意識を高め、デジタルセキュリティのためのツールを提供し、情報を行動に移すために人々を動員するという Tactical Tech のミッションにおいて、セキュリティトレーニングの取り組み、カリキュラム、組織セキュリティを主導しました。
Matt は、著名なセキュリティ研究者、オペレーショナルセキュリティトレーナー、およびデータジャーナリストであり、アッパーマンハッタンのアフリカ系アメリカ人コミュニティに基本的な暗号化ツールを教える即席ワークショップ CryptoHarlem を開催しています。
