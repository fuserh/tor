社群向来是 Tor 的心核力量，基于此信念而创立了 Tor Project 赞助会员计划，借此与众多使用 Tor 技术或认同 Tor 理念的民间营利或非营利事业组织，创建互助合作的关系。

成为会员

加入 Tor Project 会员计划以展现你对在线隐私的决心以及愿意更深入地投入 Tor 社群。加入方式请先寄信到giving@torproject.org。

红葱头会员

「对 Mullvad 而言，社群与组织的贡献是促进隐私改善与在线诚信的重要推力，不幸这样的案例很少。理解隐私问题，努力地改善指纹识别防御，保护用户不受先进攻击的人士或团队更是少之又少。我们相信 Tor Project 正是这类少数团体之一，我们共享相同的价值观诸如人权、言论自由、反审查与网络隐私。」

展现您对线上隐私的支持

当你的组织加入成为 Tor Project 会员时，不只是对在线隐私权的肯定，同时对于 Tor 的开发也会有所助益。因此基于互惠原则，我们也会提供三样特别的礼遇：顾问团队可以协助将 Tor 技术集成进你的产品、可以参加我们内核团队的内部会议以掌握目前项目的最新情况、也可借由会员身份争取更多曝光度。 若有兴趣成为会员的话，请通过giving@torproject.org与我们联系

会员等级

不同级别的会员只有在广宣曝光度上有不同程度的差异，但不论是何种级别的会员，都可以享有相同的礼遇： 顾问团队可以协助将 Tor 技术集成进您的产品中、可以参加我们内核团队的内部会议以掌握目前项目的最新情况。在 Tor 世界里，我们热爱所有的洋葱，然而对于红葱头是特别偏爱！

红葱头会员

每年赞助超过100,000美金
