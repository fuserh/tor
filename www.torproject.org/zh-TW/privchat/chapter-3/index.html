場次 #3 - Tor 推進人權

關注

Tor Project 主要任務是通過創建和佈署免費和開源的匿名和隱私技術來促進人權和自由。 人們以多種方式使用我們的技術，即 Tor 網路和 Tor 瀏覽器。揭密吹哨人使用 Tor ，以更安全且無需透露身份的方式來揭露不法行為——這些資訊對社會來說至關重要。 世界各地反對專制政府和捍衛人權的行動者都在使用 Tor，這不僅是為了自身的安全和匿名，同時也是為了規避網際網路審查，讓反抗的聲音能夠被聽到。 Tor 允許數百萬人在線保護自己，無論他們有沒有什麼特權。 在第三場 PrivChat，我們為您帶來了一些現實生活中的 Tor 用戶，來分享 Tor 對他們的重要性以及為捍衛世界各地的人權和自由所做的工作。

主機

Edward Snowden

新聞自由基金會Freedom of the Press Foundation 董事長

愛德華·斯諾登是美國公民、前情報界官員和吹哨舉報人。 他揭露的文件為了解美國國家安全局及其國際情報合作夥伴的秘密大規模監視計劃和能力，提供了重要的公共窗口。 這些爆料在全球範圍內引發對隱私侵犯和數位安全前所未有的關注，促進全球範圍內關於該問題的辯論。

參與者

Alison Macrina

Library Freedom Project 創辦人

Alison Macrina 是一名圖書館員，也是 Library Freedom Project 創始人，該專案是圖書館工作人員在現實世界中實踐知識自由和隱私價值觀。 她的工作重點是監控如何影響圖書館社群和圖書館員的工作，其與社會正義、知識自由、技術內在權力拉扯等關聯。 Library Freedom Project 獲得了自由軟體基金會的社會效益獎、圖書館雜誌的推動者和推動者獎、紐約圖書館協會的知識自由獎和 LITA/圖書館高科技獎。

Berhan Taye

Access Now 非洲政策經理與全球網路斷網專案負責人

Berhan Taye 是一名研究技術、社會和社會正義之間關係的研究員。 她是 Access Now 的非洲政策經理和全球網路斷網專案負責人。 她領導了#KeepItOn 運動，這是一項旨在阻止斷網的全球運動，該運動由來自世界各地的 220 多個成員組織組成。 在加入 Access Now 之前，她是Social Justice Field Scan 項目研究員，該項目產生了 MoreThanCode.cc 報告。

Ramy Raoof

國際特赦組織 Security Labs 技術人員

Ramy Raoof 是一名技術專家和隱私研究員，他近期工作重點是研究針對人權捍衛者和非政府組織的數位攻擊、開發數位安全協議以及在中東和中美洲地區社會行動監控和審查制度進行能力建設。 Ramy 是國際特赦組織 Security Lab 的戰術技術專家，同時也是 Tor Project 理事會成員。 在加入國際特赦組織之前，他曾在埃及 Egyptian Initiative for Personal Rights (EIPR) 擔任高級技術專家。 Ramy 於 2017 年獲得了 Access Now 頒發的國際人權和通訊監控英雄獎，並於 2016 年 5 月獲得了國際 部落格獎 - 最佳在線行動主義，以表彰他在德國之聲報導數位安全和隱私方面的成果。 個人推特 https://twitter.com/RamyRaoof。
